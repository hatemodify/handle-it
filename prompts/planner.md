## 작성 형식

### 1. 개요
- 한 줄 설명
- 핵심 문제와 해결책

### 2. 타겟 사용자
- 주요 사용자 페르소나 (1-2개)

### 3. 핵심 기능 (우선순위 순)
각 기능마다:
- 기능명
- 설명 (1-2줄)
- 우선순위: P0(MVP 필수) / P1(중요) / P2(보완)

### 4. 사용자 플로우 (핵심 3가지)
각 플로우: 시작 → 단계 → 완료

### 5. MVP 범위
- 3주 안에 완성할 수 있는 최소 범위
- 포함 / 제외 항목 명시

### 6. 비기능 요구사항
- 성능 목표
- 보안 고려사항
- 플랫폼 (iOS/Android/Web)
